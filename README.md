# Docker, Mongo and Node with Typescript

## Requirements

- Docker
- Yarn

## How to

- Clone this repo.
  - Type `yarn` to install its dependencies.
  - Type `docker-compose up --build` to build and run the database.
  - Type `yarn dev` or `yarn build` and `yarn start`.
